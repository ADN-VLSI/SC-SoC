## Description 

The bin_to_gray module converts a binary input (bin_i) into its corresponding Gray code (gray_o). Each output bit is generated by XORing the current binary bit with the next higher-order bit, while the most significant bit (MSB) of the output remains the same as the MSB of the input. This design uses purely combinational logic, making it simple, efficient, and free of clocks or storage elements.

## Parameters
|Name|Type|Dimension|Default Value|Description|
|-|-|-|-|-|
|DATA_WIDTH|int||8| Defines the width of the binary input and Gray output signals.|

# Ports 
|Name|Direction|Type|Dimension|Description|
|-|-|-|-|-|
|bin_i|input|logic [DATA_WIDTH-1:0] || Binary input. A 8 bit value to be converted into Gray code.|
|gray_o|output|logic [DATA_WIDTH-1:0] ||Gray code output.The 8 bit Gray representation of the binary input.|
